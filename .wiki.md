# Project Summary

The project is focused on creating a modern, responsive church website for Jethro Liberation Ministries Intl. It aims to provide a community-centric platform that includes essential features such as a home page, sermon archives, event management, and an admin dashboard. The website leverages technologies like Next.js, Tailwind CSS, and Supabase to deliver a user-friendly experience that caters to both church members and visitors.

# Project Module Description

- **Frontend**: Developed with Next.js and Tailwind CSS, emphasizing responsive design and smooth animations using Framer Motion. It includes various pages such as Home, About, Ministries, Sermons, Events, Giving, and Contact.
- **Backend**: Powered by Supabase for secure authentication, database management, and file storage, ensuring a scalable backend solution.
- **Admin Dashboard**: Facilitates content management for administrators, including sermons, events, and testimonials.

# File Description Inventory

- **app/backend/**: Contains backend-related documentation and configuration for Supabase.
- **app/frontend/**: Houses the frontend codebase, including components, pages, and configuration files for building the user interface.

# Technology Stack

- **Frontend**: Next.js, Tailwind CSS, TypeScript, Framer Motion
- **Backend**: Supabase (database, authentication, storage)

# Usage

1. **Install Dependencies**: Run the command to install necessary packages.
2. **Build**: Use the build command to compile the project.
3. **Run**: Start the application locally for development.
